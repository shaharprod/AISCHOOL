דוגמאות פרומפטים מפורטים
==========================

דוגמה 1: כתיבת קוד
-------------------

פרומפט:
"כתוב לי React component בשם UserCard שמקבל props:
- name (string)
- email (string)
- avatar (string, optional)
וה-component מציג כרטיס משתמש עם שם, אימייל ותמונה.
אם אין avatar, הצג אייקון default.
הוסף styling עם CSS modules."

תוצאה:
```jsx
import styles from './UserCard.module.css';

function UserCard({ name, email, avatar }) {
  return (
    <div className={styles.card}>
      <img 
        src={avatar || '/default-avatar.png'} 
        alt={name}
        className={styles.avatar}
      />
      <h3 className={styles.name}>{name}</h3>
      <p className={styles.email}>{email}</p>
    </div>
  );
}

export default UserCard;
```

דוגמה 2: תיקון שגיאה
--------------------

פרומפט:
"תקן את השגיאה בקוד הזה:
```javascript
const users = ['John', 'Jane'];
users.map(user => {
  console.log(user.name);
});
```
השגיאה היא 'Cannot read property name of string'"

תוצאה:
"הבעיה היא ש-users הוא מערך של strings, לא objects.
אם אתה רוצה להדפיס את השם, פשוט:
```javascript
users.map(user => console.log(user));
```
או אם אתה צריך objects:
```javascript
const users = [{name: 'John'}, {name: 'Jane'}];
users.map(user => console.log(user.name));
```"

דוגמה 3: שיפור קוד
-------------------

פרומפט:
"שפר את הקוד הזה:
```javascript
function getData() {
  fetch('/api/data')
    .then(res => res.json())
    .then(data => {
      console.log(data);
    });
}
```
הוסף error handling, async/await, ו-TypeScript types"

תוצאה:
```typescript
async function getData(): Promise<void> {
  try {
    const response = await fetch('/api/data');
    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Error fetching data:', error);
    throw error;
  }
}
```
