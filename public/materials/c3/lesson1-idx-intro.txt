למי השיעור?
------------
השיעור מתאים למי שרוצה לפתח אתרים או אפליקציות בסביבה שפועלת בדפדפן (בלי להתקין תוכנות כבדות), או למי שמכיר פיתוח ורוצה להכיר את Project IDX. נסביר מונחים טכניים בפשטות ונלווה שלב-אחר-שלב.

מה נלמד?
---------
• מה זה Project IDX במילים פשוטות
• איך לפתוח את IDX ולהקים פרויקט ראשון – שלב-אחר-שלב
• איך Gemini Code Assist עוזר לכתוב ולתקן קוד

באור מושגים (מילון קצר)
=========================

• Project IDX – סביבת פיתוח (עורך קוד) של Google שפועלת בדפדפן. במקום להתקין תוכנה במחשב – נכנסים לאתר ועורכים קוד בענן. מתאים לפיתוח אתרים ואפליקציות.

• סביבת פיתוח (IDE) – תוכנה או אתר שמאפשרים לכתוב קוד, להריץ ולדבג. IDX היא סביבת פיתוח בענן – הכל בדפדפן.

• Gemini Code Assist – עוזר AI שמובנה ב-IDX: מציע השלמות קוד, מסביר קוד, עוזר לתקן שגיאות. כמו "עוזר חכם" בזמן כתיבת קוד.

• ענן (Cloud) – שרתים של Google שמריצים את הסביבה. אתה לא תלוי במחשב שלך – נכנס מכל מכשיר וממשיך לעבוד.

• Deploy (פריסה) – העלאת האתר או האפליקציה לשרת כך שאחרים יוכלו לגשת אליה באינטרנט. ב-IDX יש חיבור ל-Firebase ל-Deploy מהיר.

הכוונה שלב-אחר-שלב
===================

שלב 1: פתיחת Project IDX
-------------------------
1. בדפדפן (Chrome מומלץ) גש לכתובת של Project IDX (חפש "Project IDX Google" או את הכתובת הרשמית).
2. התחבר עם חשבון Google.
3. המסך הראשי יופיע – תראה אפשרות "New project" או "פרויקט חדש".

שלב 2: יצירת פרויקט ראשון
---------------------------
1. לחץ "New project" או "Create project".
2. בחר סוג פרויקט (למשל: "Web app", "React") או "Empty" אם רוצה להתחיל ריק.
3. תן שם לפרויקט ולחץ Create – תיכנס לסביבת העריכה עם קבצי הקוד.

שלב 3: הכרת המסך
-----------------
1. במרכז – עורך הקוד (הקובץ שנפתח).
2. בצד – עץ קבצים (כל הקבצים בפרויקט).
3. למטה – טרמינל (פקודות להרצה, התקנת חבילות).
4. Gemini Code Assist – בדרך כלל מופיע כהשלמות אוטומטיות בזמן הקלדה או בחלון צד.

שלב 4: הרצה ראשונה
-------------------
1. אם בחרת תבנית (למשל React) – בדרך כלל יש פקודת הרצה כמו "npm start". בטרמינל הקלד אותה ולחץ Enter.
2. הדפדפן ייפתח עם האתר (או יוצג קישור). כך תראה שהסביבה עובדת.

השוואה ל-VS Code רגיל
=======================

📊 IDX vs VS Code

מחיר:
• IDX: ✓ חינם לחלוטין
• VS Code: ✓ חינם לחלוטין

התקנה:
• IDX: לא נדרש - עובד בדפדפן
• VS Code: נדרש התקנה על המחשב

חומרה:
• IDX: ענן - עובד על שרתי Google
• VS Code: מקומי - משתמש במחשב שלך

AI מובנה:
• IDX: ✓ Gemini Code Assist מובנה
• VS Code: Extensions בלבד (Copilot, Codeium וכו')

Preview:
• IDX: ✓ מובנה - תצוגה מקדימה אוטומטית
• VS Code: דרושים Extensions

Git:
• IDX: ✓ מובנה
• VS Code: ✓ מובנה

Extensions:
• IDX: מוגבל - רק חלק מה-Extensions
• VS Code: אלפי Extensions זמינים

עבודה צוותית:
• IDX: ✓ מובנה - שיתוף קל
• VS Code: דרושים Extensions (Live Share)

Offline:
• IDX: ✗ דורש חיבור לאינטרנט
• VS Code: ✓ עובד Offline

ביצועים:
• IDX: תלוי בחיבור לאינטרנט
• VS Code: מקסימליים - משתמש בחומרה המקומית

סימולטורים:
• IDX: ✓ מובנים (Android, iOS)
• VS Code: דורש התקנה נפרדת
