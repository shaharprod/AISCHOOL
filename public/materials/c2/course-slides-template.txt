מבנה מצגת מוצלח
=================

1. מבנה בסיסי (7-10 שקפים)
---------------------------

שקף 1: כותרת
-------------
• שם המצגת (גדול וברור)
• תת-כותרת (הקשר)
• שמך/שם החברה
• תאריך
• לוגו

תבנית Gemini:
```
צור כותרת מושכת למצגת על [נושא] לקהל [תיאור קהל],
כלול תת-כותרת שמסבירה את הערך
```

שקף 2: סקירה/Agenda
---------------------
• מה נדבר עליו
• משך הזמן
• נקודות מפתח (3-5)
• מה הקהל ירוויח

תבנית Gemini:
```
צור אג'נדה למצגת בת [X] דקות על [נושא],
כלול 4-5 נקודות מפתח
```

שקפים 3-8: תוכן
---------------
• נקודה אחת לשקף
• כותרת ברורה
• תוכן תמציתי
• ויזואליזציה
• דוגמה/מקרה בוחן

תבנית Gemini:
```
צור תוכן לשקף על [נקודה ספציפית],
כלול: כותרת, 3 נקודות תבליט, הצעה לתמונה
```

שקף 9: סיכום
-------------
• נקודות מפתח (3-5)
• מסקנות
• Call to Action
• שאלות?

תבנית Gemini:
```
סכם את הנקודות המפתח של מצגת על [נושא]
ב-3-5 bullet points, הוסף CTA
```

שקף 10: תודה וצור קשר
----------------------
• תודה על ההקשבה
• פרטי יצירת קשר
• QR code (אופציונלי)
• רשתות חברתיות

2. מבנה מורחב (15-20 שקפים)
-----------------------------

בנוסף למבנה הבסיסי:

• Problem Statement (שקף 3)
• Solution Overview (שקף 4)
• How it Works (שקפים 5-6)
• Benefits (שקפים 7-8)
• Case Studies (שקפים 9-11)
• Pricing/Plans (שקף 12)
• Testimonials (שקף 13)
• Next Steps (שקף 14)
• FAQ (שקף 15)

עיצוב עם AI
=============

1. בחירת סגנון
--------------

שאל את Gemini:
```
הצע 3 סגנונות עיצוב למצגת על [נושא] לקהל [תיאור],
כלול: צבעים, פונטים, סגנון גרפי
```

סגנונות פופולריים:
• Minimalist - נקי ומודרני
• Corporate - פורמלי ומקצועי
• Creative - צבעוני ודינמי
• Tech - עתידני וחדשני
• Elegant - מעודן ואלגנטי

2. פלטת צבעים
--------------

שאל את Gemini:
```
הצע פלטת צבעים ל[סוג עסק/נושא],
כלול: צבע ראשי, משני, הדגשה, רקע
```

טיפים:
• 3-4 צבעים מקסימום
• ניגודיות גבוהה
• עקביות בכל המצגת
• התאמה למותג

3. בחירת תמונות
---------------

שאל את Gemini:
```
הצע 5 רעיונות לתמונות לשקף על [נקודה],
תן מילות מפתח ל-לננו בננה
```

מקורות:
• לננו בננה (Google)
• Unsplash
• Pexels
• Pixabay
• תמונות עצמיות

4. אייקונים וגרפיקה
--------------------

שאל את Gemini:
```
הצע אייקונים מתאימים ל[נושאים],
תן הצעות לפריסה בשקף
```

מקורות:
• Google Icons
• Font Awesome
• Flaticon
• Noun Project

תוכן אפקטיבי
=============

1. כותרות משכות
----------------

עקרונות:
• קצרות (4-7 מילים)
• ברורות
• פעולה/תועלת
• מעוררות עניין

תבנית Gemini:
```
צור 5 כותרות משכות לשקף על [נקודה],
כל אחת עם גישה שונה
```

דוגמאות:
❌ "מידע על המוצר"
✅ "חסכו 50% מהזמן עם הכלי שלנו"

❌ "השיטה שלנו"
✅ "3 צעדים להצלחה מובטחת"

2. תוכן תמציתי
---------------

כלל 6×6:
• 6 שורות מקסימום
• 6 מילים בשורה
• נקודות תבליט
• לא משפטים ארוכים

תבנית Gemini:
```
קצר את הטקסט הזה ל-3 נקודות תבליט קצרות:
[הטקסט שלך]
```

דוגמה:
❌ "החברה שלנו פיתחה פתרון חדשני שעוזר לחברות לחסוך זמן וכסף..."
✅ 
• חיסכון של 50% בזמן
• הפחתת עלויות ב-30%
• יישום ב-24 שעות

3. סיפור מרתק
--------------

מבנה:
1. בעיה - מה הכאב?
2. פתרון - מה ההצעה?
3. תוצאות - מה ההשפעה?

תבנית Gemini:
```
צור סיפור למצגת על [מוצר/שירות],
השתמש במבנה: בעיה → פתרון → תוצאות
```

4. נתונים וסטטיסטיקה
---------------------

טיפים:
• מספר אחד גדול לשקף
• ויזואליזציה (גרף/איקון)
• הקשר ומשמעות
• מקור מהימן

תבנית Gemini:
```
הציג את הנתון [מספר] בצורה ויזואלית מרשימה,
הצע איך להדגיש בשקף
```

עבודה עם Gemini
=================

תהליך יצירה:
-------------

שלב 1: תכנון
```
עזור לי לתכנן מצגת על [נושא] ל[קהל],
צור מבנה של 8-10 שקפים עם כותרות
```

שלב 2: תוכן לכל שקף
```
צור תוכן מפורט לשקף על [כותרת השקף],
כלול: נקודות מפתח, דוגמה, הצעה לויזואליזציה
```

שלב 3: עיצוב
```
הצע עיצוב ל[שקף ספציפי],
כלול: פריסה, צבעים, תמונות
```

שלב 4: שיפור
```
שפר את השקף הזה: [הדבק תוכן],
עשה אותו יותר [מושך/תמציתי/מקצועי]
```

שלב 5: תרגול נאום
```
תן לי script לנאום לכל שקף,
כלול מעברים ונקודות הדגש
```

טיפים מקצועיים
=================

1. כלל הזהב
------------
💡 שקף אחד = רעיון אחד
💡 פחות זה יותר
💡 תמונה שווה 1000 מילים

2. טיפוגרפיה
-------------
• גופן בכותרת: 44-54pt
• גופן בגוף: 24-32pt
• משפחת פונטים: 2 מקסימום
• Contrast גבוה

3. מעברים
----------
• עקביים בכל המצגת
• לא מסחררים
• Fade או None
• זמן קצר (0.3-0.5 שניות)

4. אנימציות
------------
• רק אם מוסיף ערך
• אחת לשקף
• פשוטות
• מתוזמנות

5. נגישות
----------
• Contrast ratio 4.5:1+
• פונטים ברורים
• לא רק על צבע
• טקסט חלופי לתמונות

שגיאות נפוצות
==============

❌ יותר מדי טקסט
✅ 6 שורות מקסימום

❌ צבעים רבים
✅ 3-4 צבעים

❌ תמונות באיכות נמוכה
✅ רזולוציה גבוהה

❌ אנימציות מסחררות
✅ פשוטות או ללא

❌ פונטים קטנים
✅ 24pt מינימום

❌ חוסר עקביות
✅ תבנית אחידה

צ'ק ליסט לפני הצגה
====================

□ כל השקפים בפורמט אחיד
□ אין שגיאות כתיב
□ תמונות באיכות גבוהה
□ מספרים מדויקים
□ פונטים ברורים
□ Contrast טוב
□ זמן הנאום מתאים
□ יש גיבוי (PDF/USB)
□ נוסה על המחשב בפועל
□ יש מצב offline

פרומפטים מתקדמים
==================

1. יצירת מטאפורות
-----------------
```

הצע איך להציג בשקף
```

2. שיפור מסר
-------------
```
הפוך את המסר הזה ליותר משכנע:
[המסר שלך]
```

3. סיפור מותג
--------------
```
צור סיפור מותג למצגת שיווקית,
כלול: גיבור, אתגר, פתרון
```

4. קריאה לפעולה
----------------
```
צור 5 CTA שונים לסוף מצגת על [נושא]
```

5. טיטוי זרם
-------------
```
צור teaser למצגת על [נושא],
משהו שיגרום לאנשים לרצות לשמוע יותר
```
