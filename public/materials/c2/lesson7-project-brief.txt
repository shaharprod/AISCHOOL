הנחיות מפורטות לפרויקט הגמר - מצגת עסקית
===========================================

מבוא
----
פרויקט הגמר הוא הזדמנות ליישם את כל מה שלמדת בקורס. 
בפרויקט זה תיצור מצגת עסקית מקצועית מ-A עד Z באמצעות 
כלי Google AI.

מטרת הפרויקט
------------
ליצור מצגת עסקית מקצועית הכוללת:
- עיצוב מקצועי ואיכותי
- תוכן מעניין ורלוונטי
- תמונות שנוצרו ב-ImageFX
- שימוש ב-Gemini ליצירת תוכן
- שימוש ב-Google Slides

דרישות הפרויקט
--------------

1. נושא המצגת
-------------
בחר נושא רלוונטי למצגת עסקית:
- הצגת מוצר חדש
- אסטרטגיה עסקית
- דוח ביצועים
- הצעת שירות
- פרויקט חדש

2. מבנה המצגת
-------------
המצגת חייבת לכלול:
- שקף פתיחה (כותרת, תת-כותרת, שם המרצה)
- 8-12 שקפי תוכן עיקרי
- שקף סיכום
- שקף שאלות

3. עיצוב
--------
המצגת חייבת לכלול:
- עיצוב מקצועי ועקבי
- פלטת צבעים עקבית
- טיפוגרפיה מתאימה
- תמונות איכותיות

4. תמונות
---------
המצגת חייבת לכלול:
- לפחות 5 תמונות שנוצרו ב-ImageFX
- תמונות רלוונטיות לתוכן
- תמונות איכותיות ומקצועיות

5. תוכן
-------
המצגת חייבת לכלול:
- תוכן מעניין ורלוונטי
- שימוש ב-Gemini ליצירת תוכן
- נקודות ברורות וקצרות
- שפה מקצועית

שלבי הפרויקט
-------------

שלב 1: תכנון (יום 1)
---------------------
1. בחר נושא למצגת
2. הגדר את המטרה והקהל
3. צור מבנה בסיסי למצגת
4. תכנן את התוכן

שימוש ב-Gemini:
"עזור לי לתכנן מצגת עסקית על [נושא] עם:
- מטרה: [מטרה]
- קהל יעד: [קהל]
- מסר מרכזי: [מסר]"

שלב 2: יצירת תוכן (יום 2)
-------------------------
1. השתמש ב-Gemini ליצירת תוכן
2. צור תוכן לכל שקף
3. ערוך את התוכן
4. ודא שהתוכן רלוונטי

שימוש ב-Gemini:
"צור לי תוכן למצגת עסקית על [נושא] עם:
- שקף פתיחה
- 10 שקפי תוכן עיקרי
- שקף סיכום
כל שקף צריך כותרת ו-3 נקודות"

שלב 3: יצירת תמונות (יום 3)
-----------------------------
1. צור פרומפטים ל-ImageFX
2. צור תמונות רלוונטיות
3. עבד את התמונות
4. הוסף את התמונות למצגת

שימוש ב-ImageFX:
"צור תמונת רקע לשקף פתיחה על [נושא], 
סגנון מקצועי, צבעים [צבעים]"

שלב 4: עיצוב (יום 4)
----------------------
1. פתח מצגת חדשה ב-Google Slides
2. הוסף את התוכן
3. הוסף את התמונות
4. עצב את המצגת

שימוש ב-Google Slides:
- בחר תבנית מתאימה
- הוסף תוכן
- הוסף תמונות
- ערוך את העיצוב

שלב 5: עריכה ושיפור (יום 5)
-----------------------------
1. ערוך את התוכן
2. שפר את העיצוב
3. בדוק את המצגת
4. תרגל את המצגת

1. תכנון טוב
- תכנן את הפרויקט מראש

- צור לוח זמנים

2. שימוש בכלים
- השתמש ב-Gemini ליצירת תוכן
- השתמש ב-ImageFX ליצירת תמונות
- השתמש ב-Google Slides לעיצוב

3. איכות
- ודא שהתוכן איכותי
- ודא שהתמונות איכותיות
- ודא שהעיצוב מקצועי

4. עקביות
- שמור על עקביות בעיצוב
- שמור על עקביות בתוכן
- שמור על עקביות בסגנון

5. תרגול
- תרגל את המצגת
- בדוק את המצגת
- שפר את המצגת
