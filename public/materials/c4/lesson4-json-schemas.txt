אוסף Schema מוכנים לשימוש
===========================

1. Schema לניתוח סנטימנט
==========================

```json
{
  "type": "object",
  "properties": {
    "sentiment": {
      "type": "string",
      "enum": ["positive", "negative", "neutral"]
    },
    "confidence": {
      "type": "number",
      "minimum": 0,
      "maximum": 1
    },
    "key_phrases": {
      "type": "array",
      "items": {"type": "string"}
    },
    "emotions": {
      "type": "array",
      "items": {
        "type": "string",
        "enum": ["joy", "sadness", "anger", "fear", "surprise"]
      }
    }
  },
  "required": ["sentiment", "confidence"]
}
```

שימוש:
```python
model = genai.GenerativeModel(
    model_name="gemini-1.5-flash",
    generation_config={
        "response_mime_type": "application/json",
        "response_schema": sentiment_schema
    }
)
```

שימוש כללי
===========

דוגמת קוד לשימוש:
```python
import google.generativeai as genai
import json

genai.configure(api_key="YOUR_API_KEY")

# בחר Schema
schema = {...}  # אחד מה-Schemas למעלה

model = genai.GenerativeModel(
    model_name="gemini-1.5-flash",
    generation_config={
        "response_mime_type": "application/json",
        "response_schema": schema
    }
)

# יצירת תוכן
response = model.generate_content("השאלה או הבקשה שלך")

# פרסור התוצאה
data = json.loads(response.text)

# שימוש בנתונים
print(data)
```

טיפים
======

1. התאם Schema לצרכים שלך
2. הוסף/הסר שדות לפי צורך
3. השתמש ב-enums לערכים קבועים
4. הגדר required fields נכון
5. בדוק את ה-JSON לפני שימוש
