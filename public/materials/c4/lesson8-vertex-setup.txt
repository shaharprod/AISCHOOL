הגדרת חיבור ל-Vertex AI - צעד אחר צעד
=======================================

מדריך מפורט להגדרה
====================

שלב 1: יצירת פרויקט GCP
========================

1. היכנס ל-Google Cloud Console
--------------------------------
https://console.cloud.google.com

2. צור פרויקט חדש
------------------
• לחץ על "Select a project"
• לחץ על "New Project"
• תן שם לפרויקט
• לחץ על "Create"

3. הפעל Vertex AI API
----------------------
• חפש "Vertex AI API"
• לחץ על "Enable"

שלב 2: יצירת Service Account
==============================

1. נווט ל-Service Accounts
----------------------------
• IAM & Admin > Service Accounts
• לחץ על "Create Service Account"

2. הגדר Service Account
------------------------
• שם: vertex-ai-service
• תיאור: Service account ל-Vertex AI
• לחץ על "Create and Continue"

3. הענק הרשאות
---------------
• Role: Vertex AI User
• לחץ על "Continue"
• לחץ על "Done"

4. צור Key
-----------
• לחץ על ה-Service Account
• לחץ על "Keys" > "Add Key" > "Create new key"
• בחר JSON
• לחץ על "Create"
• שמור את הקובץ במקום מאובטח

שלב 3: הגדרת Python
====================

1. התקן ספריות
---------------
```bash
pip install google-cloud-aiplatform
```

2. הגדר Environment Variable
-----------------------------
```bash
export GOOGLE_APPLICATION_CREDENTIALS="path/to/service-account-key.json"
```

או ב-Python:
```python
import os
os.environ["GOOGLE_APPLICATION_CREDENTIALS"] = "path/to/service-account-key.json"
```

שלב 4: קוד בסיסי
=================

```python
import vertexai
from vertexai.generative_models import GenerativeModel

# אתחל Vertex AI
vertexai.init(
    project="your-project-id",
    location="us-central1"
)

# צור מודל
model = GenerativeModel("gemini-1.5-pro")

# יצירת תוכן
response = model.generate_content("מה זה AI?")
print(response.text)
```

שלב 5: בדיקה
=============

```python
def test_vertex_ai():
    """בדוק חיבור ל-Vertex AI"""
    try:
        vertexai.init(
            project="your-project-id",
            location="us-central1"
        )
        
        model = GenerativeModel("gemini-1.5-pro")
        response = model.generate_content("טסט")
        
        print("✓ חיבור הצליח!")
        print(f"תשובה: {response.text}")
        return True
    except Exception as e:
        print(f"✗ שגיאה: {e}")
        return False

# הרץ בדיקה
test_vertex_ai()
```

טיפים לפתרון בעיות
====================

בעיה: Authentication Error
----------------------------
פתרון:
• ודא ש-GOOGLE_APPLICATION_CREDENTIALS מוגדר נכון
• בדוק שה-Service Account יש לו הרשאות
• ודא שה-API מופעל

בעיה: Project Not Found
-------------------------
פתרון:
• בדוק את ה-project ID
• ודא שאתה מחובר לפרויקט הנכון
• בדוק הרשאות

בעיה: API Not Enabled
-----------------------
פתרון:
• הפעל את Vertex AI API
• המתן כמה דקות
• נסה שוב

======

✓ הגדרת קוד
✓ בדיקה
