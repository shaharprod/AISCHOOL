שיעור 10: פרויקט גמר
======================

מטרת הפרויקט
=============

בניית צ'אטבוט מלא מקצה לקצה עם כל התכונות שנלמדו בקורס.

דרישות בסיסיות
===============

1. צ'אטבוט פונקציונלי
-----------------------
• שיחה עם משתמש
• שמירת הקשר
• תשובות רלוונטיות

2. System Instructions
-----------------------
• פרסונה מוגדרת
• סגנון תקשורת
• פורמט תשובות

3. Configuration
-----------------
• Temperature מתאים
• Max tokens
• Safety settings

4. תכונות מתקדמות
------------------
• עבודה עם תמונות (אופציונלי)
• JSON Mode (אופציונלי)
• Error handling
• Logging

דרישות טכניות
==============

1. שפת תכנות
--------------
• Python או JavaScript
• קוד נקי ומתועד
• Best practices

2. תשתית
----------
• API endpoint
• Error handling
• Logging
• Basic UI (אופציונלי)

3. אבטחה
---------
• API Key מאובטח
• Input validation
• Rate limiting (אופציונלי)

מבנה הפרויקט
==============

1. Backend
-----------
• API server
• Integration עם Gemini
• Error handling
• Logging

2. Frontend (אופציונלי)
------------------------
• ממשק משתמש
• תצוגת שיחה
• Input form

3. Configuration
-----------------
• Environment variables
• Settings file
• Documentation

ציון הפרויקט
=============

1. פונקציונליות (40%)
------------------------
• צ'אטבוט עובד
• תכונות בסיסיות
• תכונות מתקדמות

2. קוד (30%)
-------------
• איכות קוד
• תיעוד
• Best practices

3. תכונות מתקדמות (20%)
--------------------------
• System Instructions
• Configuration
• Error handling
• Logging

4. תיעוד (10%)
----------------
• README
• Comments
• Documentation

טיפים להצלחה
==============

1. התחל פשוט
-------------
• בנה MVP ראשון
• הוסף תכונות בהדרגה
• בדוק בכל שלב

2. תיעוד
----------
• תיעד את הקוד
• כתוב README
• הסבר החלטות

3. בדיקות
-----------
• בדוק עם דוגמאות
• Handle edge cases
• Test error scenarios

4. שיפור
----------
• Refactor קוד
• Optimize ביצועים
• שפר UX
