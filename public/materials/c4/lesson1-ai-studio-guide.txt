למי השיעור?
------------
השיעור מתאים למי שרוצה לבדוק פרומפטים מתקדמים, להגדיר מודלים ופרמטרים, או להתחיל לעבוד עם Gemini API (לבנות אפליקציות שמשתמשות ב-Gemini). נסביר מונחים כמו API, System Instructions ופרמטרים בפשטות ונלווה שלב-אחר-שלב.

מה נלמד?
---------
• מה זה Google AI Studio ומה ההבדל בינו לבין Gemini הרגיל
• איך לפתוח AI Studio, ליצור פרומפט ראשון ולקבל מפתח API – שלב-אחר-שלב
• איך להגדיר System Instructions (הוראות קבועות למודל) ופרמטרים בסיסיים

באור מושגים (מילון קצר)
=========================

• Google AI Studio – אתר של Google (aistudio.google.com) לעבודה מקצועית עם מודלי Gemini: בדיקת פרומפטים, השוואת מודלים, והנפקת מפתח API לשימוש בתוכנות.

• Gemini (הממשק הרגיל) – gemini.google.com – לשיחה יומיומית עם AI. AI Studio – לכלי מקצועי, בדיקות והנפקת API.

• API – ממשק שמאפשר לתוכנה או לאתר לשלוח בקשות ל-Gemini ולקבל תשובות. מתאים כשאתה בונה אפליקציה, בוט או שירות שמשתמש ב-AI.

• מפתח API (API Key) – קוד סודי שמזהה אותך כשאתה שולח בקשות ל-API. מקבלים אותו ב-AI Studio; לא לחשוף בציבור או להעלות ל-Git.

• System Instructions – הוראות קבועות שאתה נותן למודל (למשל: "ענה תמיד בעברית, כעוזר כתיבה"). ב-AI Studio אפשר להגדיר ולשמור אותן.

• פרמטרים – הגדרות כמו טמפרטורה (יצירתיות), אורך מקסימלי של תשובה. ב-AI Studio אפשר לשנות אותן ולבדוק את ההשפעה.

הכוונה שלב-אחר-שלב
===================

שלב 1: פתיחת Google AI Studio
-------------------------------
1. בדפדפן גש ל-aistudio.google.com והתחבר עם חשבון Google.
2. המסך הראשי יופיע – תראה אפשרות "Create new prompt" או "Get API key" ו/או רשימת פרומפטים קיימים.

שלב 2: יצירת פרומפט ראשון
---------------------------
1. לחץ "Create new prompt" או "New" (או דומה).
2. בשדה הפרומפט כתוב למשל: "הסבר בקצרה מה זה בינה מלאכותית".
3. לחץ "Run" או "Submit" – התשובה תופיע מתחת.
4. אם יש שדה "System instruction" – כתוב שם למשל: "ענה תמיד בעברית, בקצרה."

שלב 3: קבלת מפתח API (אם צריך לבנות אפליקציה)
-----------------------------------------------
1. בתפריט או במסך חפש "Get API key" או "API keys".
2. לחץ "Create API key" ובחר פרויקט (או צור חדש).
3. העתק את המפתח – תשתמש בו בקוד או ב-.env. אל תשתף אותו ולא תעלה ל-Git.

שלב 4: שינוי פרמטרים (אופציונלי)
---------------------------------
1. חפש "Parameters" או "Model settings" – טמפרטורה, אורך מקסימלי של תשובה.
2. שנה ערך (למשל טמפרטורה מ-0.7 ל-0.3) והרץ שוב – תראה איך התשובה משתנה.

עבודה עם Prompts
==================

1. Freeform Prompt
------------------
פרומפט חופשי פשוט:

דוגמה:
```
Write a product description for a smart watch
that tracks fitness and monitors health.
```

מתאים ל:
• שאלות פשוטות
• משימות חד-פעמיות
• ניסויים מהירים

2. Structured Prompt
--------------------
פרומפט מובנה עם משתנים:

דוגמה:
```
Write a {{style}} product description 
for a {{product}} that {{features}}.

Variables:
- style: "professional"
- product: "smartphone"
- features: "has great camera"
```

מתאים ל:
• תבניות חוזרות
• Multiple variations
• A/B Testing

3. Chat Prompt
--------------
שיחה עם הקשר:

דוגמה:
```
User: What is Python?
Model: Python is a programming language...

User: Give me an example
Model: Here's a simple example...
```

מתאים ל:
• צ'אטבוטים
• שיחות ארוכות
• הקשר רצוף

הגדרת פרמטרים
===============

1. Temperature
--------------
שולט על יצירתיות:

0.0 - דטרמיניסטי (תמיד אותה תשובה)
0.5 - מאוזן
1.0 - יצירתי
2.0 - מאוד יצירתי (אקראי)

מתי להשתמש:
• 0.0-0.3: משימות עובדתיות, קוד
• 0.7-1.0: כתיבה יצירתית
• 1.5-2.0: רעיונות, סיעור מוחות

2. Top-P (Nucleus Sampling)
----------------------------
מגביל מגוון הטוקנים:

0.1 - מגוון קטן (חזוי)
0.5 - מאוזן
0.95 - מגוון רחב (מגוון)

3. Top-K
--------
מספר הטוקנים שנבחרים:

1 - הכי סביר
10 - מאוזן
40 - מגוון רחב

4. Max Output Tokens
---------------------
אורך מקסימלי של התשובה:

• 100 - תשובות קצרות
• 1000 - פסקה
• 8192 - מסמך

Multimodal Prompts
==================

עבודה עם תמונות
----------------

העלאת תמונה:
1. לחץ "Add Image"
2. בחר קובץ
3. כתוב פרומפט

דוגמה:
```
Image: [תמונה של חתול]
Prompt: "Describe this image in detail"
```

עבודה עם וידאו
---------------

העלאת וידאו:
1. לחץ "Add Video"
2. בחר MP4
3. המתן לעיבוד
4. כתוב שאלה

דוגמה:
```
Video: [סרטון של מתכון]
Prompt: "Summarize the cooking steps"
```

עבודה עם אודיו
---------------

העלאת קובץ שמע:
1. לחץ "Add Audio"
2. בחר MP3/WAV
3. כתוב בקשה

דוגמה:
```
Audio: [הקלטת פגישה]
Prompt: "Transcribe and summarize"
```
