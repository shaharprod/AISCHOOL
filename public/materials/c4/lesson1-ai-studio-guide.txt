שיעור 1: מבוא ל-Google AI Studio
==================================

מהו Google AI Studio?
=====================

Google AI Studio הוא הכלי המקצועי לעבודה עם מודלי Gemini.
זהו סביבת פיתוח ובדיקה עבור מפתחים ואנשי מקצוע.

הבדל בין Gemini ל-AI Studio:
-----------------------------

Gemini (gemini.google.com):
• ממשק משתמש פשוט
• לשימוש יומיומי
• אינטראקציות בסיסיות
• חינמי

AI Studio (aistudio.google.com):
• כלי מקצועי למפתחים
• יכולות מתקדמות
• בדיקת API
• הגדרות מפורטות
• ניהול פרויקטים

למי זה מיועד?
==============

✓ מפתחים
✓ Data Scientists
✓ Researchers
✓ Product Managers
✓ אנשי AI/ML
✓ מי שרוצה ליצור אפליקציות AI

7. Get Code
-----------
קבלת קוד מוכן:

שפות:
• Python
• JavaScript
• Java
• Go
• Kotlin
• Swift
• REST API (cURL)

דוגמת Python:
```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

response = model.generate_content("Hello!")
print(response.text)
```

עבודה עם Prompts
==================

1. Freeform Prompt
------------------
פרומפט חופשי פשוט:

דוגמה:
```
Write a product description for a smart watch
that tracks fitness and monitors health.
```

מתאים ל:
• שאלות פשוטות
• משימות חד-פעמיות
• ניסויים מהירים

2. Structured Prompt
--------------------
פרומפט מובנה עם משתנים:

דוגמה:
```
Write a {{style}} product description 
for a {{product}} that {{features}}.

Variables:
- style: "professional"
- product: "smartphone"
- features: "has great camera"
```

מתאים ל:
• תבניות חוזרות
• Multiple variations
• A/B Testing

3. Chat Prompt
--------------
שיחה עם הקשר:

דוגמה:
```
User: What is Python?
Model: Python is a programming language...

User: Give me an example
Model: Here's a simple example...
```

מתאים ל:
• צ'אטבוטים
• שיחות ארוכות
• הקשר רצוף

Gemini 1.5 Pro:

• מתאים למשימות פשוטות

הגדרת פרמטרים
===============

1. Temperature
--------------
שולט על יצירתיות:

0.0 - דטרמיניסטי (תמיד אותה תשובה)
0.5 - מאוזן
1.0 - יצירתי
2.0 - מאוד יצירתי (אקראי)

מתי להשתמש:
• 0.0-0.3: משימות עובדתיות, קוד
• 0.7-1.0: כתיבה יצירתית
• 1.5-2.0: רעיונות, סיעור מוחות

2. Top-P (Nucleus Sampling)
----------------------------
מגביל מגוון הטוקנים:

0.1 - מגוון קטן (חזוי)
0.5 - מאוזן
0.95 - מגוון רחב (מגוון)

3. Top-K
--------
מספר הטוקנים שנבחרים:

1 - הכי סביר
10 - מאוזן
40 - מגוון רחב

4. Max Output Tokens
---------------------
אורך מקסימלי של התשובה:

• 100 - תשובות קצרות
• 1000 - פסקה
• 8192 - מסמך

Multimodal Prompts
==================

עבודה עם תמונות
----------------

העלאת תמונה:
1. לחץ "Add Image"
2. בחר קובץ
3. כתוב פרומפט

דוגמה:
```
Image: [תמונה של חתול]
Prompt: "Describe this image in detail"
```

עבודה עם וידאו
---------------

העלאת וידאו:
1. לחץ "Add Video"
2. בחר MP4
3. המתן לעיבוד
4. כתוב שאלה

דוגמה:
```
Video: [סרטון של מתכון]
Prompt: "Summarize the cooking steps"
```

עבודה עם אודיו
---------------

העלאת קובץ שמע:
1. לחץ "Add Audio"
2. בחר MP3/WAV
3. כתוב בקשה

דוגמה:
```
Audio: [הקלטת פגישה]
Prompt: "Transcribe and summarize"
```
