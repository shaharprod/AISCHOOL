דוגמאות קונפיגורציה מעשיות
=============================

דוגמה 1: צ'אטבוט תמיכה טכנית
==============================

מטרה: לענות על שאלות טכניות בצורה מדויקת ומפורטת

קונפיגורציה:
```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel(
    model_name="gemini-1.5-flash",
    generation_config={
        "temperature": 0.3,
        "top_p": 0.6,
        "top_k": 20,
        "max_output_tokens": 2000,
        "stop_sequences": ["\n\n---"]
    },
    system_instruction="""אתה עוזר טכני מנוסה.
תמיד תן תשובות מדויקות ומפורטות.
השתמש בדוגמאות קוד כשצריך.
היה ידידותי ומקצועי."""
)

response = model.generate_content("איך מתקינים Python?")
print(response.text)
```

למה זה עובד:
• Temperature נמוך = דיוק
• Top-P/Top-K מאוזנים = תשובות טובות
• Max tokens גבוה = תשובות מפורטות
• System instruction = התנהגות עקבית
