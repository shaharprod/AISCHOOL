מה זה טוקן?
============

הגדרה:
------
טוקן = יחידת טקסט בסיסית
• מילה אנגלית ≈ 1-1.5 טוקנים
• מילה עברית ≈ 2-3 טוקנים
• תו מיוחד ≈ 1 טוקן

דוגמאות:
---------
"Hello" = 1 טוקן
"Hello world" = 2 טוקנים
"שלום עולם" = 4-6 טוקנים
"Hello, world!" = 4 טוקנים (כולל פיסוק)

חישוב מהיר:
-----------
• 1,000 מילים אנגלית ≈ 1,300 טוקנים
• 1,000 מילים עברית ≈ 2,500 טוקנים
• 1 עמוד A4 ≈ 500-800 טוקנים

----------------

💡 שמור תשובות נפוצות:

-------------------

]

• Less timeouts
• פחות בזבוז

תכנון תקציב
============

שלב 1: הערך צריכה
-------------------

שאלות:
• כמה requests/יום?
• כמה tokens ממוצע?
• איזה מודל?

דוגמה:
1,000 users × 10 messages = 10,000 requests/day

שלב 2: חשב עלות
----------------

נוסחה:
```
Cost = (Input_tokens × Input_price) + 
       (Output_tokens × Output_price)
```

דוגמה:
```
Input: 10,000 × 50 = 500K tokens
Output: 10,000 × 150 = 1.5M tokens

Flash:
$0.075 × 0.5 + $0.30 × 1.5 = $0.4875/day
$0.4875 × 30 = $14.625/month
```

שלב 3: הוסף buffer
-------------------

💡 הוסף 20-30% buffer:
$14.625 × 1.25 = **$18.28/month**

שלב 4: נטר וכוונן
------------------

💡 בדוק שימוש שבועי
💡 התאם תחזית
💡 אופטימיזציה

מחשבון עלות
=============

פייתון:
-------
```python
def calculate_cost(
    input_tokens: int,
    output_tokens: int,
    model: str = "flash"
):
    prices = {
        "flash": {"input": 0.075/1e6, "output": 0.30/1e6},
        "pro": {"input": 1.25/1e6, "output": 5.00/1e6}
    }
    
    cost = (
        input_tokens * prices[model]["input"] +
        output_tokens * prices[model]["output"]
    )
    
    return round(cost, 6)

# שימוש
cost = calculate_cost(10000, 5000, "flash")
print(f"${cost}")  # $0.002250
```
