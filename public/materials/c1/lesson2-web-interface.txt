מדריך מלא: ממשק Gemini בווב
==============================

שיעור 2: התמצאות בממשק Gemini
================================

סקירת הממשק הראשי
==================

1. חלון השיחה (Chat Window)
----------------------------
זהו האזור המרכזי שבו מתנהלת השיחה עם Gemini.

מבנה השיחה:
• הודעות שלך - מופיעות בצד ימין עם רקע כחול בהיר
• תשובות Gemini - מופיעות בצד שמאל עם רקע לבן
• חותמת זמן - מתחת לכל הודעה
• כפתורי פעולה - מתחת לכל תשובה

כפתורי פעולה בתשובות:
📋 העתק - מעתיק את התשובה
🔊 האזן - מקריא את התשובה בקול
👍 👎 משוב - דירוג התשובה
⚠️ דווח - דיווח על תוכן לא הולם

2. שדה הקלט (Input Field)
--------------------------
נמצא בתחתית המסך.

תכונות:
• הקלדת טקסט חופשי
• תמיכה ב-RTL (עברית/ערבית)
• גודל משתנה אוטומטית
• הדגשת תחביר לקוד
• ספירת תווים

כפתורים נוספים:
📎 צירוף קבצים (תמונות, PDF, מסמכים)
🎤 הקלטה קולית (בגרסאות מסוימות)
🔘 שליחה (או Enter)

3. סרגל צד שמאלי
------------------

אזור זה מכיל:

א) היסטוריית שיחות (Chat History)
• כל השיחות שלך מהימים האחרונים
• חיפוש מהיר בשיחות
• הוספה למועדפים ⭐
• מחיקת שיחות 🗑️
• שיתוף שיחות 🔗

טיפ: השתמש בכפתור "שיחה חדשה" כדי להתחיל נושא חדש

ב) תפריט ראשי
☰ תפריט המבורגר:
• הגדרות ⚙️
• עזרה ומשוב 💬
• שפות 🌐
• מצב כהה/בהיר 🌓
• יציאה מהחשבון

4. כותרת עליונה
-----------------

משמאל לימין:
• לוגו Google Gemini
• "שיחה חדשה" +
• חיפוש 🔍
• אייקון החשבון 👤

5. אזור הודעות המערכת
-----------------------

מופיע מעל לשדה הקלט:
⚠️ התראות על מגבלות
ℹ️ טיפים שימושיים
✨ תכונות חדשות

תכונות מתקדמות
================

1. העלאת קבצים
---------------

סוגי קבצים נתמכים:
✓ תמונות: JPG, PNG, GIF, WEBP
✓ מסמכים: PDF, DOCX, TXT
✓ טבלאות: XLSX, CSV
✓ קוד: PY, JS, HTML, CSS
✓ וידאו: MP4 (גרסאות מסוימות)

הגבלות:
• גודל מקסימלי: 20MB
• מספר קבצים: עד 10 בו-זמנית
• תמיכה ב-OCR לטקסט בתמונות

איך להעלות:
1. לחץ על 📎
2. בחר "העלה מהמחשב" או "מ-Google Drive"
3. בחר קובץ
4. כתוב שאלה לגבי הקובץ

דוגמאות לשימוש:
• "מה כתוב במסמך הזה?"
• "סכם את התמונה"
• "תרגם את הטקסט בתמונה"
• "הסבר את הקוד"

2. הקלטה קולית
----------------

זמין במכשירים ניידים ובדפדפנים מסוימים.

איך להשתמש:
1. לחץ על 🎤
2. דבר בבירור
3. לחץ "עצור" או שתוק לכ-3 שניות
4. הטקסט יתומלל אוטומטית

טיפים לדיוק טוב:
• דבר בקצב רגיל
• הקפד על סביבה שקטה
• בטא מילים בבירור
• השתמש בפיסוק ("נקודה", "פסיק")

3. ייצוא תשובות
-----------------

כפתור "Export" מאפשר:
📄 Export to Docs - יצירת מסמך ב-Google Docs
📊 Export to Sheets - יצירת טבלה
📧 Draft in Gmail - טיוטה במייל
🔗 Create shareable link - קישור לשיתוף

4. המשך שיחה
--------------

Gemini זוכר את ההקשר של השיחה:
• התייחסויות קודמות
• נתונים שסיפקת
• החלטות שקיבלת

דוגמה:
👤 "מה הבירה של צרפת?"
🤖 "הבירה של צרפת היא פריז"
👤 "ומה האוכלוסייה שלה?" ← Gemini יבין שאתה שואל על פריז

5. Double-Check (אימות עובדות)
--------------------------------

תכונה ייחודית ל-Gemini!

איך זה עובד:
• כפתור "🔍 Check" מופיע בתשובות עובדתיות
• לחיצה עליו מאמתת את העובדות דרך Google Search
• משפטים מאומתים מסומנים בירוק ✓
• משפטים לא אמינים בכתום ⚠️

מתי להשתמש:
✓ מידע עובדתי חשוב
✓ תאריכים ומספרים
✓ מקורות היסטוריים
✓ נתונים מדעיים

קיצורי מקלדת
=============

Windows/Linux:
• Ctrl + Enter - שלח הודעה
• Ctrl + K - חיפוש בהיסטוריה
• Ctrl + N - שיחה חדשה
• Ctrl + / - עזרה
• Ctrl + Shift + C - העתק עם פורמט

Mac:
• Cmd + Enter - שלח הודעה
• Cmd + K - חיפוש
• Cmd + N - שיחה חדשה
• Cmd + / - עזרה

כללי:
• Enter - שורה חדשה
• Shift + Enter - שליחה (הפוך)
• Esc - ביטול פעולה
• Tab - השלמה אוטומטית

הגדרות חשובות
==============

גישה להגדרות: ⚙️ > Settings

1. שפה וממשק
-------------
• שפת ממשק: עברית, אנגלית, ועוד
• שפת תשובות: זיהוי אוטומטי או ידני
• כיוון טקסט: RTL/LTR

2. פרטיות
----------
• שמירת היסטוריה: כן/לא
• שיתוף נתונים עם Google
• מחיקה אוטומטית: אף פעם / לאחר 3 חודשים / לאחר 18 חודשים

3. מראה
--------
• מצב כהה/בהיר/אוטומטי
• גודל פונט
• צפיפות תצוגה

4. התראות
-----------
• התראות על תשובות
• התראות מייל
• עדכונים על תכונות חדשות

5. ברירות מחדל
---------------
• אורך תשובה: תמציתי / מאוזן / מפורט
• סגנון כתיבה: פורמלי / רגיל / יצירתי
• כלול מקורות: תמיד / לפי בקשה

טיפים לשימוש יעיל
==================

1. ארגון שיחות
--------------
💡 תן שמות משמעותיים לשיחות
💡 קבץ נושאים דומים
💡 מחק שיחות ישנות שלא נחוצות
💡 שמור חשובות במועדפים ⭐

2. מקסום מהירות
-----------------
💡 השתמש בקיצורי מקלדת
💡 התחל שיחה חדשה לנושא חדש
💡 העתק תשובות ישירות ל-Docs/Gmail
💡 שמור פרומפטים נפוצים

3. איכות תשובות
-----------------
💡 היה ספציפי בשאלות
💡 תן הקשר מלא
💡 בקש דוגמאות
💡 שאל שאלות המשך
💡 השתמש ב-Double-Check

4. אבטחה
---------
⚠️ אל תשתף מידע רגיש
⚠️ השתמש במצב פרטי לנושאים רגישים
⚠️ בדוק הגדרות פרטיות
⚠️ מחק שיחות עם מידע אישי

בעיות נפוצות ופתרונות
======================

בעיה: Gemini לא מגיב
פתרון:
1. רענן את הדף (F5)
2. בדוק חיבור לאינטרנט
3. נסה דפדפן אחר
4. נקה cache וcookies
5. בדוק ב-status.google.com

בעיה: לא ניתן להעלות קובץ
פתרון:
1. בדוק סוג הקובץ
2. בדוק גודל (מקס 20MB)
3. נסה להמיר פורמט
4. העלה מ-Google Drive

בעיה: תשובות לא מדויקות
פתרון:
1. נסח את השאלה מחדש
2. תן יותר הקשר
3. פצל לשאלות קטנות יותר
4. השתמש ב-Double-Check
5. בקש מקורות

בעיה: ממשק בשפה לא נכונה
פתרון:
1. ⚙️ > Settings > Language
2. בחר עברית
3. רענן דף
