קטגוריה 1: פרודוקטיביות (10 טריקים)
=====================================

1. שמור פרומפטים מוצלחים
-------------------------
שמור פרומפטים שעובדים טוב לשימוש חוזר.

דוגמה:
שמור פרומפט: "סכם את המסמך הזה ב-3 פסקאות"
→ השתמש שוב ושוב

2. יצירת תבניות
---------------
צור תבניות לפרומפטים נפוצים.

דוגמה:
תבנית: "כתוב [סוג תוכן] על [נושא] ל-[קהל]"
→ מלא את החסר לפי הצורך

3. שימוש בקיצורי מקלדת
-----------------------
למד קיצורי מקלדת לחיסכון זמן:
• Ctrl/Cmd + Enter - שליחה
• Ctrl/Cmd + K - חיפוש
• Ctrl/Cmd + N - שיחה חדשה

4. ייצוא מהיר
-------------
השתמש בייצוא ישיר:
• Export to Docs
• Export to Sheets
• Draft in Gmail

5. שימוש בהיסטוריה
------------------
השתמש בהיסטוריה למציאת שיחות קודמות.

6. חיפוש בהיסטוריה
-------------------
השתמש בחיפוש למציאת שיחות ספציפיות.

7. שמירה במועדפים
-----------------
שמור שיחות חשובות במועדפים ⭐

8. שיתוף שיחות
---------------
שתף שיחות עם אחרים 🔗

9. העתקה מהירה
---------------
השתמש בכפתור העתקה מהירה 📋

10. יצירת קישורים
------------------
צור קישורים לשיתוף שיחות.

קטגוריה 2: איכות תשובות (10 טריקים)
=====================================

11. היה ספציפי
--------------
ככל שתהיה יותר ספציפי, התשובות יהיו טובות יותר:
✓ "סכם את המסמך ב-3 פסקאות ממוקדות"
✗ "סכם את המסמך"

12. תן הקשר מלא
---------------
תמיד תן הקשר מלא:
• רקע
• מטרה
• קהל יעד

13. בקש דוגמאות
---------------
בקש דוגמאות ספציפיות:
"תן דוגמאות ל-[נושא]"

14. שאל שאלות המשך
------------------
השתמש בתשובות כבסיס לשאלות נוספות.

15. בקש הסברים
---------------
בקש הסברים מפורטים:
"הסבר את [נושא] בפירוט"

16. בקש שיפורים
---------------
בקש שיפורים:
"שפר את [תוכן]"

17. השתמש ב-Double-Check
------------------------
השתמש ב-Double-Check למידע חשוב.

18. בקש מקורות
-------------
בקש מקורות למידע:
"מה המקורות ל-[מידע]?"

19. בקש חלופות
-------------
בקש מספר אפשרויות:
"תן לי 3 אפשרויות ל-[נושא]"

20. בקש תובנות
-------------
בקש תובנות עמוקות:
"מה התובנות החשובות?"

קטגוריה 3: עבודה עם קבצים (10 טריקים)
=======================================

21. העלאה מ-Drive
-----------------
השתמש בהעלאה מ-Drive לגישה ישירה.

22. העלאה מרובה
---------------

23. ניתוח לפי סוג קובץ

התאם את הפרומפט לסוג הקובץ:
• PDF - ניתוח מסמך
• תמונה - ניתוח תמונה
• Excel - ניתוח נתונים

24. חילוץ מידע ספציפי
---------------------
בקש חילוץ מידע ספציפי:
"הוצא את [סוג מידע] מהקובץ"

26. יצירת סיכומים
-----------------
בקש סיכומים מפורטים:
"צור סיכום מפורט של הקובץ"

27. תרגום קבצים
---------------
בקש תרגום קבצים:
"תרגם את הקובץ הזה ל-[שפה]"

28. עריכת קבצים
---------------
בקש עריכת קבצים:
"ערוך את הקובץ הזה"

29. יצירת תוכן מקבצים
--------------------
בקש יצירת תוכן חדש מקבצים:
"צור [סוג תוכן] על בסיס הקבצים האלה"

30. שילוב קבצים
---------------
שלב מספר קבצים לניתוח משולב.
