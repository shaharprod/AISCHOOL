ניתוח מסמכים ו-PDF ארוכים
============================

עבודה עם מסמכים מורכבים
=========================

📄 יכולות Gemini למסמכים
===========================

מה ניתן לעשות:
---------------
✓ סיכום מסמכים (עד 1M טוקנים!)
✓ חילוץ מידע
✓ ניתוח תוכן
✓ תרגום
✓ השוואת מסמכים
✓ שאלות ותשובות

סוגי קבצים נתמכים:
-------------------
• PDF
• DOC/DOCX
• TXT
• Images (עם טקסט)
• Spreadsheets

📊 אסטרטגיות לניתוח
=====================

1. סיכום מהיר
--------------
"סכם את המסמך ב-3 נקודות עיקריות"

2. סיכום מפורט
---------------
"סכם כל פרק בנפרד עם נקודות מפתח"

3. חילוץ מידע
--------------
"מצא את כל התאריכים, שמות, ומספרים במסמך"

4. ניתוח נושאים
----------------
"מה הנושאים המרכזיים? ציין תדירות"

5. השוואה
----------
"השווה בין מסמך 1 למסמך 2"

🎯 מקרי שימוש
==============

חוזים משפטיים
--------------
• "סכם את תנאי החוזה"
• "מה הסעיפים הקריטיים?"
• "מצא התחייבויות"
• "זהה סיכונים"

דוחות מחקר
-----------
• "מה הממצאים העיקריים?"
• "סכם את המתודולוגיה"
• "מה המסקנות?"
• "רשום ציטוטים חשובים"

דוחות כספיים
--------------
• "מה מצב הכספים?"
• "נתח מגמות"
• "זהה חריגות"
• "השווה לשנה שעברה"

מסמכי מכרז
-----------
• "מה הדרישות?"
• "מה קריטריוני הבחירה?"
• "מה תאריכי אבן דרך?"
• "מצא חובות"

ספרות אקדמית
--------------
• "סכם מאמר מדעי"
• "מה המחקר הראשי?"
• "ציין מקורות"
• "השווה לתיאוריות אחרות"

💡 טכניקות מתקדמות
====================

Chunking ארוך
--------------
למסמך ארוך מאוד:
1. חלק למקטעים
2. סכם כל מקטע
3. סכם את הסיכומים

Chain of Thought
-----------------
"נתח את המסמך צעד אחר צעד:
1. זהה נושא ראשי
2. מצא תת-נושאים
3. רשום נקודות מפתח
4. סכם מסקנות"

Multi-doc Analysis
------------------
"השווה את 3 המסמכים:
• מה משותף?
• מה שונה?
• מה הסתירות?"

🔍 שאלות חכמות
===============

בסיסי:
------
• "מה הנושא הראשי?"
• "כמה עמודים?"
• "מי כתב את זה?"

ניתוח:
------
• "מה הטיעון המרכזי?"
• "מה הראיות?"
• "מה החולשות?"

השוואה:
--------
• "מה השתנה מגרסה קודמת?"
• "איך זה מתיישב עם [מסמך אחר]?"

יישום:
------
• "איך אני יכול להשתמש בזה?"
• "מה הצעדים הבאים?"
• "מה הסיכונים?"

📋 תבניות פרומפטים
====================

לסיכום:
--------
```
סכם את המסמך בפורמט הבא:
1. רקע (2-3 משפטים)
2. נקודות עיקריות (5 bullets)
3. מסקנה (1 משפט)
```

לחילוץ מידע:
-------------
```
חלץ מהמסמך:
- שמות אנשים
- תאריכים חשובים
- סכומי כסף
- מיקומים
החזר כטבלה
```

לניתוח:
--------
```
נתח את המסמך לפי:
1. SWOT (חוזקות, חולשות, הזדמנויות, איומים)
2. סיכון (גבוה/בינוני/נמוך)
3. המלצות (3-5 נקודות)
```

⚡ טיפים למהירות
=================

1. היה ספציפי
--------------
❌ "ספר לי על המסמך"
✅ "מה 3 הנקודות הכי חשובות בעמוד 5?"

2. פרמט התוצאה
---------------
✅ "החזר כטבלה"
✅ "רשום בנקודות"
✅ "מספר כל פריט"

3. הגבל אורך
-------------
✅ "סכם ב-100 מילים"
✅ "תן 5 נקודות בלבד"

4. שאל בשלבים
--------------
✅ שלב 1: "מה הנושא?"
✅ שלב 2: "ספר יותר על X"
✅ שלב 3: "תן דוגמה"

🎓 תרגילים
===========

תרגיל 1:
---------
העלה PDF → סכם ב-5 נקודות

תרגיל 2:
---------
2 מסמכים → השווה ותן המלצה

תרגיל 3:
---------
חוזה → זהה סיכונים

תרגיל 4:
---------
דוח → צור מצגת מהנתונים
