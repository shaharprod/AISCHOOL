רשימת פרומפטים מוכנים ל-Gmail
=================================

50+ פרומפטים מוכנים לשימוש
============================

📧 ניהול מיילים עם Gemini
===========================

מבוא
====

רשימה זו כוללת 50+ פרומפטים מוכנים לשימוש עם Gemini ב-Gmail.
כל פרומפט ניתן להעתיק ולהדביק ישירות!

איך להשתמש:
-----------
1. פתח את Gmail
2. לחץ על "Help me write" ✨
3. העתק את הפרומפט
4. התאם לצרכים שלך
5. לחץ Enter

קטגוריה 1: תגובות מהירות
===========================

1. אישור קבלת מייל
-------------------
```
כתוב תגובה קצרה ומקצועית שמאשרת קבלת המייל ואומרת שאטפל בנושא בהקדם
```

2. תודה על מידע
----------------
```
כתוב מייל תודה חם ומקצועי למידע שקיבלתי, הדגש את התועלת שבמידע
```

3. בקשה לדחיית פגישה
--------------------
```
כתוב מייל מנומס שמבקש לדחות פגישה ליום אחר, הציע 3 תאריכים חלופיים
```

4. אישור פגישה
--------------
```
כתוב אישור פגישה קצר עם פרטי המפגש: [תאריך], [שעה], [מקום]
```

5. המלצה על שירות
-----------------
```
כתוב מייל המלצה חמה על [שם] שעבד איתי ב[פרויקט], הדגש את [תכונות]
```

קטגוריה 2: בקשות ופניות
=========================

6. בקשת מידע נוסף
-----------------
```
כתוב מייל מקצועי שמבקש מידע נוסף על [נושא], ציין מדוע זה חשוב
```

7. פנייה ללקוח חדש
------------------
```
כתוב מייל היכרות ללקוח חדש, הצג את [החברה/עצמך], ציע לקבוע שיחת היכרות
```

8. בקשת הצעת מחיר
-----------------
```
כתוב מייל מקצועי שמבקש הצעת מחיר ל[שירות/מוצר], ציין דרישות ולוח זמנים
```

9. פנייה למומחה
---------------
```
כתוב מייל מכובד לפרופסור/מומחה, בקש ייעוץ ב[נושא], הצע זמני פגישה
```

10. בקשת שיתוף פעולה
--------------------
```
כתוב הצעת שיתוף פעולה ל[חברה/אדם], הדגש יתרונות הדדיים, הצע פגישה
```

קטגוריה 3: תשובות מורכבות
===========================

11. טיפול בתלונה
----------------
```
כתוב תגובה אמפתית ומקצועית לתלונת לקוח על [בעיה], הצע פתרונות קונקרטיים
```

12. דחייה מנומסת
----------------
```
כתוב מייל דחייה מנומס להצעה/בקשה, הסבר את הסיבה בעדינות, השאר דלת פתוחה
```

13. הבהרת אי הבנה
-----------------
```
כתוב מייל שמבהיר אי הבנה שקרתה ב[נושא], הצג את העובדות בצורה ברורה ומכבדת
```

14. עדכון על עיכוב
------------------
```
עדכן על עיכוב ב[פרויקט/משלוח], הסבר את הסיבה, הצע פתרון ולוח זמנים חדש
```

15. הודעת התפטרות
-----------------
```
כתוב מייל התפטרות מקצועי וחם, תודה על ההזדמנות, הצע תקופת מעבר
```

קטגוריה 4: מיילים שיווקיים
============================

16. הזמנה לאירוע
-----------------
```
כתוב הזמנה מושכת לאירוע [שם האירוע] ב[תאריך], כלול פרטים ו-CTA ברור
```

17. עדכון מוצר חדש
------------------
```
כתוב מייל שמציג מוצר/שירות חדש, הדגש יתרונות, כלול הצעה מיוחדת
```

18. תזכורת לפעולה
-----------------
```
כתוב תזכורת ידידותית ל[פעולה] שטרם בוצעה, הדגש את החשיבות
```

19. מבצע מוגבל בזמן
-------------------
```
כתוב מייל מושך למבצע מוגבל בזמן ל[מוצר/שירות], צור תחושת דחיפות
```

20. סקר שביעות רצון
-------------------
```
כתוב בקשה למילוי סקר שביעות רצון, הסבר מדוע זה חשוב, הבטח שזה קצר
```

קטגוריה 5: מיילים פנימיים
===========================

21. עדכון לצוות
---------------
```
כתוב עדכון שבועי לצוות על [פרויקט], כלול הישגים, אתגרים וצעדים הבאים
```

22. תיאום ישיבה
--------------
```
כתוב מייל לתיאום ישיבת צוות, הצע 3 זמנים, כלול נושאים לדיון
```

23. ברכה לעובד
--------------
```
כתוב ברכה חמה לעובד על [הישג/יום הולדת], הדגש את התרומה שלו
```

24. בקשת עזרה מעמית
-------------------
```
בקש עזרה מעמית ב[נושא ספציפי], הסבר מה צריך, תודה מראש
```

25. שיתוף הצלחה
---------------
```
שתף הצלחה של הצוות ב[פרויקט], תודה לתורמים, חזק את רוח הקבוצה
```

קטגוריה 6: מיילים רשמיים
==========================

26. מכתב המלצה
--------------
```
כתוב מכתב המלצה מקצועי ל[שם] על [תפקיד], הדגש כישורים והישגים
```

27. הצעת עבודה
--------------
```
כתוב הצעת עבודה רשמית ל[שם] לתפקיד [תפקיד], כלול תנאים ותאריך התחלה
```

28. הודעת סיום עבודה
--------------------
```
כתוב הודעת סיום עבודה רשמית אך מכבדת, ציין סיבה ותקופת הודעה
```

29. תלונה רשמית
---------------
```
כתוב תלונה רשמית על [נושא], ציין עובדות, דרוש פעולה, שמור טון מקצועי
```

30. הסכם שיתוף פעולה
--------------------
```
כתוב הסכם שיתוף פעולה עם [חברה], ציין תנאים, זכויות וחובות
```

קטגוריה 7: מעקבים (Follow-ups)
================================

31. מעקב אחרי פגישה
-------------------
```
כתוב מעקב אחרי פגישה מ[תאריך], סכם החלטות, ציין משימות וצעדים הבאים
```

32. מעקב אחרי הצעה
------------------
```
עקוב אחרי הצעה ששלחתי ב[תאריך], שאל האם יש שאלות, הצע שיחה
```

33. תזכורת ידידותית
-------------------
```
שלח תזכורת ידידותית ל[משימה/תשלום], אל תהיה לחוץ, הצע עזרה
```

34. מעקב אחרי אירוע
-------------------
```
שלח תודה למשתתפים ב[אירוע], שתף חומרים, בקש משוב
```

35. עדכון סטטוס
---------------
```
עדכן על סטטוס [פרויקט/בקשה], ציין אחוזי התקדמות, הערך זמן סיום
```

קטגוריה 8: ניוזלטרים ועדכונים
===============================

36. ניוזלטר חודשי
-----------------
```
כתוב ניוזלטר חודשי עם: עדכוני חברה, טיפים, תוכן מעניין, CTA
```

37. עדכון שינויים
-----------------
```
הודע על שינויים ב[מדיניות/שירות], הסבר את הסיבות, ציין תאריך יישום
```

38. טיפ שבועי
-------------
```
שלח טיפ שבועי ב[תחום], הסבר יישום מעשי, כלול דוגמה
```

39. סיכום רבעוני
----------------
```
סכם רבעון: הישגים, אתגרים, לקחים, יעדים לרבעון הבא
```

40. הכרזה על שותפות
-------------------
```
הכרז על שותפות חדשה עם [חברה], הסבר יתרונות ללקוחות
```

קטגוריה 9: מיילים אישיים
==========================

41. ברכה לחג
-------------
```
כתוב ברכת [חג] חמה ואישית, כלול איחולים ותודה על שנה משותפת
```

42. תנחומים
-----------
```
כתוב הודעת תנחומים רגישה ומכבדת, הצע תמיכה ועזרה
```

43. הזמנה פרטית
---------------
```
הזמן חברים/קולגות ל[אירוע פרטי], ציין פרטים, בקש אישור הגעה
```

44. עדכון שינוי כתובת
---------------------
```
הודע על שינוי כתובת/טלפון, ציין פרטים חדשים ותאריך תחולה
```

45. בקשת המלצה
--------------
```
בקש המלצה מ[מנהל/עמית], ציין למה זה נחוץ, הצע נקודות למיקוד
```

קטגוריה 10: טיפים מתקדמים
===========================

46. סיכום שרשור ארוך
--------------------
```
סכם את השרשור הזה בנקודות תבליט, ציין החלטות ומשימות שנקבעו
```

47. תרגום מייל
--------------
```
תרגם את המייל הזה לעברית/אנגלית, שמור טון מקצועי ונימוס
```

48. הצעת כותרת
--------------
```
הצע 5 כותרות משכות למייל על [נושא], כל אחת עם גישה שונה
```

49. בדיקת טון
-------------
```
בדוק את המייל הזה, האם הטון מתאים ל[מטרה]? הצע שיפורים
```

50. שכתוב פורמלי
----------------
```
שכתב את המייל הזה בטון פורמלי/בלתי פורמלי יותר, שמור את התוכן
```

טיפים לשימוש מתקדם
====================

1. התאמה אישית
--------------
💡 תמיד התאם את הפרומפט למצב הספציפי
💡 הוסף פרטים רלוונטיים
💡 שנה את הטון לפי הנמען

2. שילוב פרטים
--------------
💡 החלף [סוגריים] בפרטים אמיתיים
💡 הוסף שמות, תאריכים, מקומות
💡 ציין הקשר ספציפי

3. בדיקה ועריכה
---------------
💡 תמיד קרא את התוצאה
💡 ערוך בהתאם לצורך
💡 בדוק איות ודקדוק

4. שמירת פרומפטים
------------------
💡 שמור פרומפטים שעובדים טוב
💡 צור ספרייה אישית
💡 שתף עם הצוות

5. שיפור מתמיד
--------------
💡 נסה וריאציות
💡 למד מתוצאות
💡 התאם לסגנון שלך

דוגמאות שימוש
===============

דוגמה 1: תגובה מהירה
---------------------
לפני:
```
Hi, got your email about the project.
Will look into it.
Thanks
```

עם Gemini:
```
כתוב תגובה קצרה ומקצועית שמאשרת קבלת המייל על הפרויקט
ואומרת שאטפל בנושא בהקדם
```

אחרי:
```
שלום,

תודה על הפנייה לגבי הפרויקט. קיבלתי את המייל ואטפל בנושא
במהלך היום. אעדכן אותך בהקדם.

בברכה
```

דוגמה 2: מייל שיווקי
---------------------
לפני:
```
We have a new product. Check it out.
```

עם Gemini:
```
כתוב מייל שמציג מוצר חדש לניהול משימות עם AI,
הדגש חיסכון בזמן, כלול הצעה מיוחדת ל-30 יום ניסיון חינם
```

אחרי:
```
נושא: חסכו 5 שעות שבועיות עם כלי הניהול החדש שלנו 🚀

היי [שם],

רציתי לשתף אותך במשהו מרגש!

השקנו זה עתה כלי ניהול משימות חדש המשלב בינה מלאכותית
שעוזר לחסוך עד 5 שעות עבודה בשבוע.

מה מיוחד?
✓ תזמון אוטומטי חכם
✓ סיכום פגישות ב-AI
✓ תזכורות מותאמות אישית

הצעה מיוחדת: 30 יום ניסיון חינם!
[לחץ כאן להתחלה]

נשמח לשמוע מה אתה חושב,
[שמך]
```

שגיאות נפוצות
==============

❌ פרומפט כללי מדי
✅ ציין פרטים ספציפיים

❌ לא התאמת את הטון
✅ ציין את הטון הרצוי (רשמי/בלתי רשמי)

❌ שכחת להוסיף הקשר
✅ ציין למי, מדוע, מתי

❌ לא בדקת את התוצאה
✅ תמיד קרא ועדכן לפני שליחה

❌ לא שמרת פרומפטים טובים
✅ שמור ושפר את הספרייה שלך
